Strings need to be in double quotes: “101011”

Integers are left as is: 1

Expression can be either a string, or another command/function

Some nested commands only work with certain others, all of them work with star product

Basic functions

Shift Maximality - results will be either 1 (is shift maximal) or -1 (is not shift maximal)

sm(expression)

sm(“101011”)

-1

Cutting Times

ct(expression)

ct(“101011”)

1,2,6

Word Count - returns how many subwords of a certain length are in the main string

wc(expression, integer)

wc(“101011”, 2)

{01=2, 10=2, 11=1}

Star Product - creates a new string based on certain rules

sp(expression, expression)

sp(“101011”, “101011”)

101011110101101010111101011010101111010111101011

Concatenation - adds the second string on to the the end of the first string

concat(expression, expression)

concat(“101011”, “101”)

101011101

Insertion - insert after an index

insert(expression, expression, index)

insert(“101011”, “111”, 2)

101111011

Substitution - substitute characters of a string with different characters

sub(expression, “string”->”string”)

sub(expression, “string”->”string”,“string”->”string”)

sub(“101”, “1”->”0”,”0”->”1”)

010

Compare - compare two strings and gives the position in which they differ

cmp(string, string)

cmp(“101011”, “010”)

1 2 3 4 5 6

Assigning variables

variable_name := “string”

x := “101011”

Checking variable assignment

print variable_name or variable_name

print x

x  

Nested Commands

All regular commands can have other commands in them

sp(sp(expression, expression), expression)

sp(sp(“101011”, “101”), “010”)

wc(sp(expression, expression), integer)

wc(sp(“101011”, “101”), 3)

Console Commands

/help

Prints out a list of all console commands and a short help string for each

/exit

Quits the application.

/clear

Clears all text in the output window.

/exec <file path>

Executes a batch of several sequential commands.  To automatically insert a file path into the command line, use Ctrl + I, or Edit > Insert File Path... from the menu bar.

Batch files should be written in text format (.txt).  Below is an example.

a := "1"

b := "0"

c := sm(sp(a, b))

print c

/debug

Toggles developer debug mode on and off.  Not recommended for normal use.
